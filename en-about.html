How to make a Tokyo proxy

How to create with render　


1、Go to Render.com(Log in to github and render)
2，Press "new+" and press "Web Service"
3、Enter “https://github.com/konomi-tokyo/tokyo-proxy-V3.git” in the Public Git repository frame
4、Decide on a domain name and press “Create Web Service”
5、Wait until "your service is live"
6、Complete！


How to create in sandbox


1、Go to Codesandbox(Link Github and codesandbox)
2、Press "Import repository"
3、Enter "https://github.com/konomi-tokyo/tokyo-proxy-V3.git" in the GitHub Repository URL location and press "import"
4、Press fork repository
5、Type “npm run start” in the box at the bottom of the screen and press Enter.
6、Complete！
