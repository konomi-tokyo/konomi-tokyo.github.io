東京プロキシとは

東京プロキシは、小中学生を対象に学校で配布されたタブレット(学タブ)の規制を回避するサイトです。
東京プロキシを使えばYouTubeやDiscord、Pixivなどの様々なサイトにアクセスできます。
初期バージョンは、現在使用できません。V2以降は別サイトと偽っており、ブロックされにくくなっています。

作り方

Renderで作成する方法 (翻訳を使うと詰む可能性あり)


1、Render.comに行く(githubとrenderにログインしておく)
2，「new＋」を押して「Web Service」を押す
3、Public Git repositoryの枠に「https://github.com/konomi-tokyo/tokyo-proxy-V3.git」を入れる
4、ドメイン名を決めて　「Create Web Service」　を押す
5、「your service is live」まで待つ
6、完成！
※学タブ(i-filter)勢の人は不確定サイトでブロックされるのを防ぐために、「your service is live」になってから 　8~16時間後までは稼働させとく


Sandboxで作成する方法


1、Codesandboxに行く(Githubとcodesandboxを連携させとく)
2、Import repositoryを押す
3、GitHub Repository URLの場所に「https://github.com/konomi-tokyo/tokyo-proxy-V3.git」を入力し、importを押す
4、Fork repositoryを押す
5、画面下の枠に「npm run start」と入力しエンター
6、完成！
